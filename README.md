# Few-Shot-NeRFs

this is a repo for implementation real world NeRF architectures into NeRFStudio. In specfic, these models have a main focus of using small amounts of samples for their training. That's why I called them FewShot NeRFs (that is not a literature standard naming, the proper naming is often near "Generalizable NeRFs", but I will try to make it the new standard....). I have to finish these impl in a week and train them all in LLFF with subsampling (process which will happen on another repo). So, stay tunned

a NeRFStudio unnoficial implementation for: 
- [ ] GNT (WIP)
- [ ] pixelNeRF
- [ ] VolRecon
- [ ] IBRNet
- [ ] Zero-1-to-3

this repo (each impl) will later be added into PyPi for those who want to import and use it alongside you default nerfstudio env
